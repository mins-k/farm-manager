# 밤농장 장부 MVP

가족 밤농장의 작업, 비용, 수확, 판매를 기록하는 MVP입니다.

## 포함 기능

- 농장 위치: 청수리, 양사리1, 양사리
- 작업 기록: 방제, 예초, 전지, 비료, 벌통, 수확, 출하, 식재 등
- 비용 기록: 인건비, 점심값, 간식값, 농약, 비료, 전지비, 묘목비 등
- 수확 기록: kg이 아니라 가마 단위
- 판매 기록: 유통업자, 농협, 산림조합
- 연도별 총비용, 총매출, 순이익, 재고 가마 수

## 실행 방법

### 1. 백엔드 실행

```bash
cd backend
npm install
npm run dev
```

백엔드 주소:

```txt
http://localhost:4000
```

### 2. 프론트엔드 실행

새 터미널에서:

```bash
cd frontend
npm install
npm run dev
```

프론트 주소:

```txt
http://localhost:5173
```
